背了很多泰語單字卻講不出來？只有 input 沒有 output 的問題

單字本背了一輪又一輪，看泰劇、聽泰語歌也聽得懂七八成，可是真的遇到泰國人要開口講話，腦袋卻一片空白，一句完整的話都擠不出來。如果你也有這種感覺，問題通常不是背得不夠多，而是你的學習只有「輸入」，沒有「輸出」。

「聽得懂」跟「講得出來」是兩種不同的能力

背單字、看影片、聽 podcast，這些都是輸入（input）——資訊進到你的腦袋裡，你能辨認、能理解。但輸出（output）是另一回事：你要主動從腦中把單字、文法組織起來，講成一句完整、對方聽得懂的話，而且是在幾秒鐘之內就要反應出來，沒有時間慢慢想。

這兩種能力用的腦部路徑不一樣。只練輸入，你的大腦習慣的是「被動辨認」；但講話需要的是「主動組織」，這個能力不練，就算你認得再多單字，臨場還是會卡住、找不到詞、語序亂掉。

為什麼只輸入很難進步到會說話

  • 大腦沒有「必須現在反應」的壓力：看影片聽不懂可以暫停重播，但對話沒有暫停鍵，這種即時反應的能力只能靠實際開口練出來。
  • 不知道自己哪裡卡住：一直輸入，你以為自己都懂了；但實際開口才會發現，某個文法、某個單字你其實只是「看得懂」，不是「用得出來」。
  • 沒有真實的犯錯機會：語言學習很依賴「犯錯－被糾正－記住」這個循環。光輸入沒有輸出，你不會犯錯，當然也就沒有這個循環，卡住的地方永遠卡住。

怎麼從輸入轉成輸出？

  1. 學完就立刻造句、講出來：背完一個單字，不要只是在心裡默唸，試著用它造一句跟自己生活有關的句子，講出聲音。
  2. 找真人對話，哪怕只有簡單幾句：跟老師、語言交換對象，哪怕只是打招呼、聊今天吃了什麼，都是真正的輸出練習。
  3. 錄音回聽自己的發音：輸出不是只有「講出來」，還要能發現自己講的跟正確答案差在哪，這一步最好搭配聽得出差異的人（例如母語者或老師）幫你確認。
  4. 用造句遊戲練習組織句子的反應速度：如果暫時找不到對話對象，也可以先從「限時組出正確句子」開始練習，訓練大腦主動組織語言的肌肉記憶。

找老師的最大差別：逼你輸出，而且馬上告訴你哪裡卡住

自己一個人很難逼自己開口，也很難分辨「我剛剛講的到底對不對」。一對一課最直接的價值，就是每一堂課都是強制輸出練習——老師會問你問題、等你回答，你講錯了，當下就被糾正，這個循環自學很難靠自己完成。
